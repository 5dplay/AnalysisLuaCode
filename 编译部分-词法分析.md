# 终结符

语法分析通过词法分析器获取对应的终结符，Lua的终结符有：

```
//关键字
and, break, do, else, elseif, end, false, for, function,
goto, if, in, local, nil, not, or, repeat, return, then, true,
until, while

//操作符，和其他终结符
/, .., ..., ==, >=, <=, ~=, <<, >>, ::, TK_EOS(尚未知道是什么),
浮点数, 整数, 标识符, 字符串
```

# 相关结构体/联合体

|成员名|类型|说明|
|:-|:-|:-|

|i|lua_Integer||
